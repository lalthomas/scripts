* Clear the contents of that file according to the following instructions

	1.Never delete any @ symbol in this file unless specified 
	  and you should be also careful to remove any @ symbol 
	  in your program explanation.

	2.Remove the text "The Problem" with the title of 
	 the program source code

	3.Remove the text "The Quest" with the explanation 
	 of the program source code

	4.Remove the text "Foot Notes" with the significance
	 of the program( this section is optional)

	5.Remaining sections are for adding links to
	  the blog post.( this section is optional,but 
	  highly recommended)

	6.To remove any optional section,please delete 
	  that section with the remaining @ symbols

		eg. This will be a article with no external links
